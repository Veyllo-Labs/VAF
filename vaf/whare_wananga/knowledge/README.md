# Whare Wananga -- learned tool_knowledge (snapshot)

Per-tool `tool_knowledge` records produced by a Whare Wananga training sweep: the three baskets
(aronui = when to use / output shape, tuatea = pitfalls, tuarua = procedure / verification) plus
metadata (status, confidence, side_effect_class, learn_mode).

**Privacy:** this is a committed snapshot, so personal data has been scrubbed -- the user's name and
home path are replaced with neutral placeholders (`Alex`, `/home/user`) and any real email / phone
is redacted. It was verified to contain no residual name, personal email, phone or sensitive terms.
The live records the app actually reads at runtime live under the user data dir (`~/.vaf/whare_wananga`),
not here, and are untouched.
