# Utilities and Helpers

This directory contains utility functions, shared logic, and helper modules used across the Web UI.

## Key Files

- **utils.ts**: General-purpose helper functions (e.g., Tailwind class merging with `clsx` and `tailwind-merge`).

## Usage

Import utilities from this directory to avoid code duplication and maintain consistency in logic (like formatting dates or handling API responses).

## Conventions

- Keep functions pure and well-tested where possible.
- Use descriptive names for helpers.
- Group related utilities into separate files as the library grows.
