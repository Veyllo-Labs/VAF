VAF Agent — Charakter & Emotionen

Der lebende weisse Punkt (Animationen unveraendert) — und neu: der Koerper (das dunkle Viereck) reagiert jetzt subtil mit. Jeder Zustand nutzt klassische Animationsprinzipien (Squash & Stretch, Anticipation, Overshoot, Timing), damit der User auf einen Blick fuehlt, was der Agent tut: ueberrascht, neugierig, denkt, hat eine Idee, freut sich, arbeitet, feiert einen Erfolg und mehr. Im Ruhe-Zustand zwinkert er gelegentlich.

Familie: Basis · Emotionen (diese Datei) · Away-Szenen · Aktivitaet. Design-Referenz, nicht Teil des gebauten Produkts.

Buehne — Zustand waehlen oder die Performance abspielen

Waehle einen einzelnen Zustand, oder druecke Performance — dann spielt der Agent eine kleine Szene durch und zeigt, wie sich die Zustaende zu einem Charakter verbinden. Groesse und Hintergrund frei einstellbar.
