VAF Agent — Away scenes

What does the agent do when nobody is talking to it? It passes the time. These idle animations appear when the user opens an old or archived chat — the agent reads the newspaper, watches TV, takes a coffee break, juggles or stargazes.

The agent consists of two parts: the dark rounded square = its body, and the white dot = its eye / face. The props sit outside its body — it handles them (body leans & bobs, the eye looks, scans, blinks).
